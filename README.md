# Personal Portfolio Website

This is a simple personal portfolio website created as a school project.

## About
The website presents basic information about me, my skills, and personal content. It includes several pages such as a homepage, skills page, and gallery.

## Technologies Used
- HTML
- CSS

## Purpose
The goal of this project is to practice building and structuring a basic multi-page website.

## How to Run
Download the project and open `index.html` in your browser.
